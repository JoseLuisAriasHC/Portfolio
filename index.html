PreguntasMatch
Sitio Web

Este es un sitio web centrado en los Tests de opiniones, es decir, no hay respuestas correctas o incorrectas, sino más bien opiniones que la gente tiene respecto a ciertas preguntas. La dinámica de la web sera la siguiente, en un principio, habría unos tests base para las distintas categorías, de modo que el usuario pueda probarlos. Posteriormente, se alimentaría de las opiniones y tests que los usuarios vayan creando. Por cada test, se les mostrarían los resultados globales para que tengan una vision de que tan apoyada es su opinión.
